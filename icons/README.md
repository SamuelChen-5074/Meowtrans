# 图标文件说明

此文件夹需要包含以下三个图标文件：

## 所需文件

1. **icon16.png** - 16x16 像素
2. **icon48.png** - 48x48 像素  
3. **icon128.png** - 128x128 像素

## 创建图标的方法

### 方法一：使用在线工具

1. 访问 https://www.favicon-generator.org/ 或类似的图标生成网站
2. 上传您的图片或使用网站提供的模板
3. 生成并下载所需尺寸的图标文件
4. 将文件放入此文件夹

### 方法二：使用图像编辑软件

使用 Photoshop、GIMP、Figma 或其他图像编辑工具：

1. 创建新画布，尺寸为 128x128 像素
2. 设计您的图标（建议使用渐变紫色主题以匹配插件）
3. 导出为 PNG 格式
4. 调整为其他两个尺寸（48x48 和 16x16）
5. 保存所有三个文件到此文件夹

### 方法三：使用 AI 生成

您可以使用 AI 图像生成工具创建图标，然后调整尺寸。

## 临时解决方案

如果您暂时没有图标文件，可以：

1. 下载任何 PNG 图片作为临时图标
2. 使用在线工具调整尺寸
3. 重命名为所需的文件名

## 注意事项

- 确保所有图标都是 PNG 格式
- 建议使用透明背景
- 保持图标在不同尺寸下的清晰度
- 图标应该简洁明了，易于识别
