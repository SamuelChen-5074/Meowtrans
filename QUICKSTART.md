# 快速开始指南

本指南将帮助您在5分钟内开始使用 Ollama 网页翻译插件。

## 步骤 1：生成图标文件（30秒）

1. 在浏览器中打开项目根目录下的 [`generate-icons.html`](generate-icons.html:1) 文件
2. 点击 **"下载所有图标"** 按钮
3. 将下载的三个图标文件保存到 `icons` 文件夹中

## 步骤 2：安装并启动 Ollama（2分钟）

### Windows
```bash
# 1. 访问 https://ollama.com 下载并安装 Ollama
# 2. 打开命令提示符，启动服务
ollama serve

# 3. 在新的命令提示符窗口中，下载模型
ollama pull qwen2:7b
```

### macOS/Linux
```bash
# 1. 安装 Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 2. 启动服务
ollama serve

# 3. 在新的终端窗口中，下载模型
ollama pull qwen2:7b
```

**重要**：保持 `ollama serve` 终端窗口打开！

## 步骤 3：安装 Chrome 插件（1分钟）

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 开启右上角的 **"开发者模式"** 开关
4. 点击 **"加载已解压的扩展程序"**
5. 选择此项目的 `translate` 文件夹
6. 完成！

## 步骤 4：配置并测试（2分钟）

1. 点击浏览器工具栏中的插件图标（如果没有图标，点击拼图图标找到插件）
2. 确认配置：
   - Ollama 服务地址：`http://localhost:11434`
   - 模型名称：`qwen2:7b`
   - 目标语言：选择您需要的语言
   - 翻译模式：选择"选中文字"
3. 点击 **"保存设置"**

## 步骤 5：开始翻译

### 方法 1：使用插件界面
1. 打开任意网页（推荐英文网页）
2. 选中一段文本
3. 点击插件图标
4. 点击 **"翻译"** 按钮
5. 查看翻译结果

### 方法 2：使用右键菜单
1. 选中一段文本
2. 右键点击
3. 选择 **"翻译选中文字"**
4. 查看翻译结果

## 常见问题快速解决

### ❌ 插件无法连接到 Ollama
**解决**：
- 确认 `ollama serve` 正在运行
- 在浏览器中访问 `http://localhost:11434/api/tags` 测试连接

### ❌ 翻译速度很慢
**解决**：
- 使用更小的模型：`ollama pull qwen2:1.5b`
- 在插件设置中修改模型名称为 `qwen2:1.5b`

### ❌ 翻译质量不好
**解决**：
- 确保使用支持目标语言的模型
- 尝试更大的模型（如 `qwen2:7b`）
